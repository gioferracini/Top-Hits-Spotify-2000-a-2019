# 🎧 Análise dos Top Hits do Spotify (2000–2019) — Projeto Python
*Python • Pandas • Matplotlib • Análise de Dados*

![Header](Imagens/header.png)

Este projeto explora os maiores sucessos do Spotify entre 2000 e 2019, utilizando Python para identificar padrões e características comuns entre músicas populares. A análise aborda artistas, gêneros, atributos musicais e indicadores relacionados à popularidade.

O objetivo é compreender como determinados elementos musicais se relacionam com o sucesso comercial ao longo do século XXI.
